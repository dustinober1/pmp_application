# Changelog

All notable changes to PMP Study Pro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Progressive Web App (PWA) support with manifest and service worker
- Comprehensive CI/CD pipeline with automated linting, testing, and type checking
- Playwright E2E testing for critical user flows
- CodeQL security analysis
- Dependabot for automated dependency updates
- Code of Conduct and Security policy documentation
- Service worker for offline functionality and caching
- Enhanced manifest with app shortcuts and capabilities

### Changed
- Updated deployment workflow to include quality checks before deployment
- Enhanced HTML metadata with PWA support

### Fixed
- Fixed missing dev:web-svelte script

## [1.0.0] - Previous Release

### Added
- Initial SvelteKit migration
- Offline-first storage with localStorage
- Spaced repetition algorithm (SM-2)
- Question practice sessions
- Dashboard with progress visualization
- Data export/import functionality
- 1,800+ flashcards
- 1,200+ practice questions
- Study guides for all PMP domains
- Formula calculator
